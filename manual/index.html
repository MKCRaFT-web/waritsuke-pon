使い方マニュアル

iPhoneで「割付ポンっ（安定版）」を初めて使う方向けに、操作手順をまとめています。

このページは GitHub Pages で確実に開ける公開用マニュアルです。編集元の原稿は docs/user-manual.md に保持しています。

1. 準備するもの

  • iPhone（Safari 推奨）
  • 対象図面の PDF（または画像）
  • 図面上で実寸が分かる寸法線（例：1000mm）
  • 入力するタイル寸法（mm）

2. 基本操作の流れ

  1. PDF を読み込む
  2. 1. 水平合わせ で図面の傾きを補正する
  3. 2. 縮尺合わせ で実寸mmを設定する
  4. 3. 施工範囲指定 で施工エリアを囲む
  5. 4. グリッド生成（表示） → 5. グリッド確定（固定）
  6. 6. 基準点 を置いて X左 / X右 / Y上 / Y下 を確認する

※ 基準点はグリッド生成後に使用します。先に押しても進めないため、必ず手順5の後で実行してください。

3. 各ステップ詳細

3-1. PDF読込

左上のファイル選択から図面を読み込みます。下部ステータスが「図面読み込み完了」になればOKです。

3-2. 水平合わせ

水平基準にしたい直線上に2点を打って 確定 します。長い直線で合わせるほど精度が上がります。

3-3. 縮尺合わせ

寸法線上の2点を指定し、実寸値（mm）を入力して 確定 します。単位は mm で入力してください。

3-4. 施工範囲指定

施工したい内側ラインに沿って点を順番に追加します。誤タップは 戻す で修正し、完了したら 完了（範囲ロック） を押します。

3-5. グリッド生成・確定

タイル寸法（幅・高さ）を入力し、4. グリッド生成（表示） を実行します。位置調整後に 5. グリッド確定（固定） を押して確定します。

3-6. 基準点で割付確認

6. 基準点（交点スナップ） で基準点を置くと、下部に左右上下の距離（mm）や枚数目安が表示されます。

4. つまずきやすいポイント

  • 縮尺合わせ前にグリッドを出すと結果がずれます。
  • 寸法値を cm / m で入れると誤差が大きくなります（mm指定）。
  • 範囲指定の点順が飛ぶと形状が崩れます。
  • 図面を再読込すると、既存の点・範囲・縮尺はリセットされます。
