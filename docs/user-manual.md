# 割付ポンっ（iPhone向け）画像付き使い方マニュアル原稿

> 対象：iPhoneで「割付ポンっ（安定版）」を初めて使う方  
> この原稿は、**PDF読込 → 基準点合わせ → 水平合わせ → 縮尺合わせ → 範囲指定 → 割付確認**までを、初心者向けに1操作ずつ説明したものです。

---

## 1. アプリ概要

### 1-1. このアプリでできること
- 図面（PDF/画像）を読み込む
- 図面の傾きを直す（水平合わせ）
- 図面に実寸を合わせる（縮尺合わせ）
- 施工する範囲を囲む（施工範囲指定）
- タイルの割付グリッドを表示・調整する
- 基準点から左右上下の距離（mm）と枚数・半端寸法を確認する

## 2. 使う前の準備

### 2-1. 準備するもの
- iPhone（Safari推奨）
- 対象図面のPDF（または画像）
- 図面上で実寸がわかる寸法線（例：1000mm）
- 入力するタイル寸法（mm）

**何のための操作か**  
→ 後の「縮尺合わせ」と「枚数計算」を正しく行うためです。

**よくある失敗・注意点**
- 寸法値が書かれていない図面を使うと、縮尺が合わず結果がずれます。
- タイル寸法（幅・高さ）を実物と違う値で入力すると、割付結果が変わります。

[画像: 事前準備の例（PDF・寸法線・タイル寸法メモ）]  
- スクリーンショット対象：準備物がわかる説明用画像（PDFサムネイル、寸法線、タイル寸法メモ）
- 画像下の短い説明文：図面・寸法線・タイル寸法の3点を先に用意します。
- 画像注釈文：
  - 「寸法線がある図面を使う」
  - 「タイル寸法を先に確認」
- altテキスト：`作業前に必要なPDF図面、寸法線、タイル寸法メモの例。`

### 2-2. iPhone操作の基本
- 点を置く：タップ
- 点を微調整：点をタップして選択 → 十字キーで動かす
- 拡大・縮小：左下の `＋ / － / 戻す`（工程により表示）

**何のための操作か**  
→ 指操作だけで精度を上げるためです。

**よくある失敗・注意点**
- ピンチ操作前提で進めると混乱しやすいです。基本は画面内ボタンで調整します。
- 誤タップしたら、範囲指定中は `戻す` ボタンで1点ずつ戻せます。

## 3. 基本の流れ

> 依頼の順番に沿って説明します。  
> ただし実際のボタン番号は **1.水平合わせ → 2.縮尺合わせ → 3.施工範囲指定 → 4.グリッド生成 → 5.グリッド確定 → 6.基準点** です。  
> **基準点はグリッド生成後にしか置けない**ため、章4では「最終確認前の基準点合わせ」として案内します。

1. PDFを読み込む
2. 水平合わせ（図面の傾きを補正）
3. 縮尺合わせ（実寸に合わせる）
4. 施工範囲指定（施工エリアを囲む）
5. グリッド生成・調整・確定
6. 基準点を置く
7. 下部表示で割付結果を確認

## 4. 各機能の詳しい使い方

### 4-1. PDF読込

#### 手順1：`ファイル選択`でPDFを読み込む
- 左上のファイル入力欄（`ファイルを選択`）をタップ
- iPhoneのファイルアプリからPDFを選ぶ
- 下部表示が `図面読み込み完了` になればOK

**何のための操作か**  
→ 割付の元になる図面をキャンバスに表示するためです。

**よくある失敗・注意点**
- 何も表示されないときは、ファイル選択が途中でキャンセルされていないか確認。
- 読み込み直しすると、前の点・範囲・縮尺はリセットされます。

[画像: PDF選択〜読込完了]  
- スクリーンショット対象：ファイル選択直後と、`図面読み込み完了` 表示の状態
- 画像下の短い説明文：読込完了メッセージが出たら次へ進みます。
- 画像注釈文：
  - 「ここをタップ：ファイル選択」
  - 「下部表示：図面読み込み完了を確認」
- altテキスト：`PDF読み込み操作と完了表示の確認画面。`

### 4-2. （依頼順）基準点合わせ ※実際は後工程

> この操作は `6. 基準点（交点スナップ）` ボタンで行い、**グリッド表示後**に使います。  
> ここでは先に「何をする機能か」を理解しておきます。

#### 手順2：基準点機能の役割を理解する
- 基準点は、割付の「原点」として扱うチェック点です。
- 置いた後、下部に `X左 / X右 / Y上 / Y下` の距離（mm）と「枚数＋半端」が表示されます。

**何のための操作か**  
→ 左右上下の取り合い寸法を、現場に近い形で確認するためです。

**よくある失敗・注意点**
- グリッド未生成のまま `6. 基準点` を押すと先に進めません。
- 交点・角から離れた場所に置くと、欲しい確認値になりにくいです。

[画像: 基準点結果の表示例（X左/X右/Y上/Y下）]  
- スクリーンショット対象：基準点設置後に下部4行が数値表示されている状態
- 画像下の短い説明文：基準点を置くと、左右上下の距離と枚数目安を確認できます。
- 画像注釈文：
  - 「ここが基準点」
  - 「下部4行を確認」
- altテキスト：`基準点設置後にX左・X右・Y上・Y下が表示された画面。`

### 4-3. 水平合わせ（1. 水平合わせ）

#### 手順3：`1. 水平合わせ` をタップ
- 左メニューの `1. 水平合わせ` を押す
- 画面が「2点指定モード」になります

**何のための操作か**  
→ 傾いた図面をまっすぐにして、後工程の誤差を減らすためです。

**よくある失敗・注意点**
- 壁ラインや通り芯など、できるだけ長い直線上で2点を取ると精度が上がります。
- 短い線分で合わせると、わずかなズレが大きくなります。

[画像: 水平合わせモード開始]  
- スクリーンショット対象：`1. 水平合わせ` を押した直後（確定ボタンが表示された状態）
- 画像下の短い説明文：まずは水平を取りたい直線上に2点を置きます。
- 画像注釈文：
  - 「① 直線上の1点目」
  - 「② 直線上の2点目」
  - 「点を置いたら確定」
- altテキスト：`水平合わせで2点を指定する画面。`

#### 手順4：2点を打って `確定`
- 水平基準にしたい線の始点・終点をタップ
- 点を打ったあとに位置を微調整したいときは、もう一度その点をタップして拡大表示に入り、十字キーで少しずつ合わせる
- 微調整が終わったら `点完了` を押して元の画面に戻る
- `確定` をタップ

**何のための操作か**  
→ 2点の傾きを使って図面全体を回転補正するためです。

**よくある失敗・注意点**
- 点を間違えたら、もう一度 `1. 水平合わせ` を押してやり直せます。
- 補正後に明らかな違和感があれば、迷わず再実行してください。

[画像: 水平合わせ確定前後比較]  
- スクリーンショット対象：確定前（斜め）と確定後（水平）
- 画像下の短い説明文：確定すると図面が回転して水平基準に揃います。
- 画像注釈文：
  - 「確定前：ラインが斜め」
  - 「確定後：ラインが水平」
- altテキスト：`水平合わせの確定前後を比較した画面。`

### 4-4. 縮尺合わせ（2. 縮尺合わせ）

#### 手順5：`2. 縮尺合わせ` をタップ
- 左メニューの `2. 縮尺合わせ` を押す
- 寸法線に沿って2点を取る準備をする

**何のための操作か**  
→ 画面上の長さを実寸mmに変換するためです。

**よくある失敗・注意点**
- 寸法線から外れた2点を取ると、面積も枚数もずれます。
- 先に水平合わせを済ませると、点が取りやすくなります。

[画像: 縮尺合わせモード開始]  
- スクリーンショット対象：`2. 縮尺合わせ` を押し、`実寸(mm)`入力欄が表示された状態
- 画像下の短い説明文：寸法線に沿って2点を取り、実寸mmを入力します。
- 画像注釈文：
  - 「寸法線の端を2点で指定」
  - 「ここに実寸を入力」
- altテキスト：`縮尺合わせ画面。2点指定と実寸入力欄が表示されている。`

#### 手順6：実寸を入力して `確定`
- 例：図面寸法が1000なら `1000` を入力
- 寸法線上に打った点をあとで微調整したいときは、もう一度その点をタップして拡大表示に入り、十字キーで位置を合わせる
- 調整が終わったら `点完了` を押して元の画面に戻る
- `確定` をタップ
- 下部に `縮尺確定：xxxx mm/px` が出ればOK

**何のための操作か**  
→ グリッドサイズ、距離表示、面積・枚数計算の基準を確定するためです。

**よくある失敗・注意点**
- 単位はmmです（cmやmで入れない）。
- 0や空欄だと確定できません。

[画像: 縮尺確定メッセージ]  
- スクリーンショット対象：`縮尺確定：... mm/px(図面)` と表示された状態
- 画像下の短い説明文：この表示が出たら縮尺設定は完了です。
- 画像注釈文：
  - 「縮尺確定メッセージを確認」
- altテキスト：`縮尺確定後にmm/pxが表示されている画面。`

### 4-5. 範囲指定（3. 施工範囲指定）

#### 手順7：`3. 施工範囲指定` をタップ
- 左メニューの `3. 施工範囲指定` を押す
- 施工したい内側ラインに沿って、順に点をタップ
- 点を置くときは、指を離さずにグリッド位置を合わせてから置くと狙った場所に合わせやすい
- 細かい位置調整は十字キーで行い、終わったら `点完了` を押して元の画面に戻る

**何のための操作か**  
→ どこにタイルを貼るかをアプリに教えるためです。

**よくある失敗・注意点**
- 点の順番が飛ぶと形が崩れます。
- 角部は少し細かめに点を置くと形が合いやすいです。

[画像: 施工範囲の点打ち中]  
- スクリーンショット対象：点を複数打って、オレンジ範囲が形成されている途中
- 画像下の短い説明文：内壁ラインをなぞるイメージで点を追加します。
- 画像注釈文：
  - 「点を順に追加」
  - 「角は細かく打つ」
  - 「間違えたら戻す」
- altテキスト：`施工範囲指定中に複数点を打っている画面。`

#### 手順8：必要に応じて `戻す`、終わったら `完了（範囲ロック）`
- 誤タップ時：`戻す`
- 範囲が完成したら：`完了（範囲ロック）`

**何のための操作か**  
→ 施工範囲を固定し、次のグリッド計算に進むためです。

**よくある失敗・注意点**
- 完了前に点数が少なすぎると正しい形になりません（最低3点以上が必要）。
- 完了後に直したい場合は `✏️ 範囲調整（点を動かす）` を使います。

[画像: 範囲ロック完了]  
- スクリーンショット対象：`施工範囲確定（範囲ロック）` の表示が出た状態
- 画像下の短い説明文：範囲がロックされたら、次はグリッド生成へ進みます。

## 5. 結果の見方

### 5-1. 下部の4項目（X左 / X右 / Y上 / Y下）
- 表示例：`1234.5mm（4枚 + 34.5mm）`
- 意味：基準点から境界までの距離、そこに入る枚数、最後の半端寸法

**何のための確認か**  
→ 現場で「どこから何枚目で何mm残るか」を事前に把握するためです。

**注意点**
- これは基準点と範囲形状に依存します。基準点を変えると値も変わります。

[画像: 下部4項目の読み方]  
- スクリーンショット対象：4項目が埋まっている状態を拡大した画像
- 画像下の短い説明文：mm値と「枚数 + 半端寸法」をセットで確認します。
- 画像注釈文：
  - 「距離(mm)」
  - 「枚数」
  - 「半端寸法(mm)」
- altテキスト：`X左X右Y上Y下の表示内訳を示した拡大画像。`

### 5-2. 積算エリア（施工面積・必要枚数・発注枚数）
- 施工面積：指定範囲の面積
- 必要枚数(理論)：ロスなし想定の枚数
- 発注枚数(ロス込)：ロス率を加味した枚数

**何のための確認か**  
→ 材料手配の目安を早く把握するためです。

## 7. トラブル時の確認ポイント

### 7-1. 画面が思ったように動かない
1. `図面ロック：ON/OFF` を確認（必要なときだけON）
2. いまの工程表示（下部の `工程:`）を確認
3. 必要なら同じ工程ボタンを押し直す（例：`1. 水平合わせ`）

### 7-2. 数値が「-」のまま
1. 縮尺確定できているか確認
2. 施工範囲が完了しているか確認
3. グリッドが表示されているか確認
4. 基準点が置かれているか確認

### 7-3. 結果に違和感がある
1. 水平合わせをやり直す
2. 縮尺合わせを寸法線上でやり直す
3. 範囲点を `✏️ 範囲調整` で修正
4. グリッドを再調整（必要なら `🎯 最適割付` も試す）

### 7-4. 最後に画像保存したい
- `📸 保存（割付図）` をタップ
- PNG画像として保存される
